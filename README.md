## Общая информация

В этом репозитории содержится серверная часть проекта для дисциплины "Визуальное программирование". Клиентская часть располагается в смежном репозитории "mobiledev."

В папке external содержатся сторонние библиотеки, необходимые для работы программы, в частности, imgui и implot. Добавлены сабмодулем.

В папке src находится код проекта, main.cpp. В нём содержится и логика общения с клиентом, и графическая оболочка программы.

Папка build внесена в .gitignore, инструкция по сборке программы приведена ниже. 

## Сборка проекта

В корне проекта:

```bash
mkdir build && cd build
cmake ../
make -j16 #16 - количество потоков. Можно ставить иное число
```

## Логика работы main.cpp

Структура TelemetryData содержит в себе всю передаваемую с телефона информацию.

```C++
struct TelemetryData {
    float lat = 0, lon = 0, alt = 0, acc = 0;
    int rsrp = 0; // основной сигнал
    string type = "";
    string raw = "";
    mutex mtx; 

    // мап для хранения истории каждой уникальной вышки
    map<string, CellHistory> cell_logs; 
    
    double base_timestamp = 0;      // время первой записи для точки отсчета
    float view_min_time = 0;        // начало диапазона просмотра
    float view_max_time = 100;      // конец диапазона просмотра
    float max_recorded_time = 100;  // общая длительность лога
} g_data;
```

## Ключевые функции

- `parse_json_to_data(string raw)`: Извлекает координаты, тип сети и проходит циклом по массиву Cells. Для каждой вышки (LTE или GSM) формируется уникальный ID на основе PCI и частоты, после чего данные добавляются в вектор истории.

 - `load_history()`: При запуске программа считывает файл telemetry_log.json и восстанавливает все графики из предыдущих сессий.

 - `zmq_server()`: Фоновый поток, принимающий JSON-пакеты по tcp. Сохраняет сырые данные в файл и обновляет текущее состояние через парсер.

 - `ui_loop()`: Графический цикл (sdl2 + OpenGL3).

 - `Smartphone Data`: Выводит текущие координаты и Net Type. Содержит слайдеры для управления временным диапазоном графика.

 - `Signal Level History`: Использует implot для отрисовки множества разноцветных линий. Каждая линия соответствует отдельной вышке. Поддерживает масштабирование и обрезку по времени через установленные фильтры.
